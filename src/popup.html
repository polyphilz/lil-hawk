GPT model (?):

As of July 9th, 2023, there are 3 main GPT models OpenAI offers for its Chat API:

  • GPT 3.5 (4k context size)
  • GPT 3.5 (16k context size)
  • GPT 4 (8k context size)

Power

Despite being faster, GPT 3.5 analyses aren't as good as GPT 4 ones. For the best results, it is recommended to use GPT 4.

Cost

That being said, GPT 4 is more expensive than GPT 3.5. Analyzing an average Terms & Conditions will cost:

  • GPT 3.5 (4k) - $0.02 USD
  • GPT 3.5 (16k) - $0.03 USD
  • GPT 4 (8k) - $0.30 USD

Context size

Context size is how many words/characters the model can work with. Specifically, the unit is "tokens." An average Terms & Conditions is ~8k tokens (e.g. the 23andMe Terms of Service). If you try to take an 8k token-length T&C and analyze it using GPT 3.5 (4k), things will break! Well not to fret; lil' hawk gets around this using some hackery, but if you're intent on using a 3.5 model, ideally use the 16k one for best results.
